由于英文不是很好，所以记录一下程序开发中遇到的一些英文词汇和短语以加深印象

## 词汇

| 英文     | 翻译 | 衍生词汇 | 短语                   |
| -------- | ---- | -------- | ---------------------- |
| scraping | 爬取 | crawler  | scraping html file     |
| crawler  | 爬虫 | scraping | search engine crawlers |

## 短语句子

| 短语               | 翻译            | 备注                 |
| ------------------ | --------------- | -------------------- |
| typescript support | typescript 支持 | 当支持某项技术时使用 |

## 技巧

- 虚拟头结点 `const dummy = new ListNode(-1);` 作为新节点的起始，最后要返回新链表时只需返回 `dummy.next`。当需要创造一条新链表的时候，可以使用虚拟头结点简化边界情况的处理。
- 如果需要把原链表的节点接到新链表上，而不是 new 新节点来组成新链表的话，那么断开节点和原链表之间的链接可能是必要的。

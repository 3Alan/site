## Units

- `rem`: Relative to the root element font size (`html/:root`) in CSS3.

   If the font size of `html` is 16px (default), then 1rem = 16px.

- `em`: Relative to its own font size if set, or relative to the parent element.

   For example, if a `div` has a font size of 10px, then 1em within that `div` is equal to 10px.

   If the element does not have a font size set and the parent element has a font size of 20px, then 1em is equal to 20px.

- `vh/vw`: Relative to the viewport. 10vh = 1/10 of the screen height.

## Difference between `translate` and absolute positioning

`translate` will occupy the original position, while absolute positioning will be taken out of the document flow.

## BFC

Concept: It is an isolated and independent container on the page, where the child elements will not affect the outside elements.

Use cases: Clearing floats, solving margin collapse.

**An element cannot exist in two BFCs at the same time.**

### Ways to create a BFC:

- Root element
- Float is not none
- `position: absolute/fixed`
- `display: inline-block/table-cell`
- Overflow is not visible
- Flexbox (`display: flex/inline-flex`)

## Clearing floats

- Pseudo-elements

```css
.clearfix::after {
  content: '';
  display: block;
  clear: both;
}
```

- Creating a BFC

```css
overflow: hidden;

overflow: auto;
```

- Empty box

```html
<div style="clear: both;"></div>
```

- Table-cell

```css
.outer {
  width: 400px;
  height: 400px;
  background: aqua;
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
.inner {
  width: 100px;
  height: 100px;
  background: brown;
  display: inline-block;
}
```

## Units

- `rem`: relative to the font-size of the root element (html/:root) css3

  If the font-size of html is 16px (default), then 1rem=16px

- `em`: If the font-size is set for the element, then relative to the element. If not set, then relative to the parent element.

  For example, if the font-size of div is set to 10px, then when em is used in the div, 1em is 10px

  If the font-size is not set for the element and the font-size of the parent element is set to 20px, then 1em is 20px

- `vh/vw` relative to the viewport, 10vh=1/10 of the screen height

## The difference between translate and using absolute positioning

translate will occupy the original position, while absolute positioning will脱离文档流.

## BFC

Concept: It is a vague and abstract isolated independent container on the page, and the child elements in the container will not affect the elements outside.

Usage scenarios: clear floats, solve margin overlap

**An element cannot exist in two BFCs at the same time**

### How to create:

- Root element
- float is not none
- position: absolute/fixed
- display: inline-block/table-cell
- overflow is not visible
- Flexbox (`display: flex/inline-flex`)

## Clear floats

- Pseudo-class elements

```css
.clearfix::after {
  content: '';
  display: blcok;
  clear: both;
}
```

- Create BFC

```css
overflow: hidden;

overflow: auto;
```

- Empty box

```html
<div style="clear: both;"></div>
```

```css

  width: 400px;
  height: 400px;
  background: aqua;
  display: table-cell;
```

基于 Docusaurus 的个人网站，主要以文档为主博客为辅

## Features

- 🔍 支持 Algolia 搜索
- 📈 支持 Google Analytics、Umami、Clarity 等多种网站分析工具，可以了解网站的流量和用户行为
- 💬 集成了 Giscus 评论功能
- 🚀 支持搜索引擎自动推送 Action，加快搜索引擎的收录

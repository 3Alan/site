### 命令

- `kubectl create deloyment [name] [image loction]` 发布应用
- `kubectl get deloyments` 列出已发布的应用
- `kubectl get pods` 列出所有 pods
- `kubectl describe pods`
- `kubectly apply -f [fileDir]` 创建服务
- `kubectl get namespace` 列出所有命名空间
- `kubectl create namespace <insert-namespace-name-here>`
